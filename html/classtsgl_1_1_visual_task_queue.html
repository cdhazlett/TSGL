Detailed Description

Provides a a visualization tool for a parallel queue of elements.

VisualTaskQueue is a self-contained Canvas intended to help visualize the progress of a parallel queue of taks or elements.

Given a maximum queue length of N elements, a VisualTaskQueue will display N rectangles on its internal Canvas. These rectangles can be in one of three states: cleared, running, or finished. Cleared elements are always displayed in white, whereas running elements are displayed in a darker shade of the current thread's color, and finished elements are displayed in a lighter shade of the color of the thread the completed the element.
