Detailed Description

Canvas extended with Cartesian drawing operations.

CartesianCanvas provides a Canvas with a Cartesian coordinate system for ease of plotting.

Note
While on a regular Canvas, pixels higher on the screen have a lower y-value, on a CartesianCanvas, pixels higher on the screen have a higher y-value.

tsgl::CartesianCanvas::CartesianCanvas ( double  timerLength = 0.0)

Default CartesianCanvas constructor method.

This is the default constructor for the CartesianCanvas class.

Parameters
timerLengthThe minimum number of seconds between draw cycles for the Canvas. A value less than or equal to 0 sets it to automatic.
Returns
A new CartesianCanvas, stretching from -400 to +400 on the x axis and -300 to +300 on the y axis, in the middle of the screen with no title. The created CartesianCanvas will take up approximately 90% of the monitor's height, and will have a 4:3 aspect ratio.

This function binds another function to the current CartesianCanvas, waits until that function is complete, and waits for the user to close the CartesianCanvas. This function effectively calls start(), myFunction(), and wait() in sequence.

Parameters
myFunctionThe function to run on the CartesianCanvas. Must take exactly one parameter of type CartesianCanvas&, which is a reference to the CartesianCanvas to render to.

void tsgl::CartesianCanvas::sleep ( )

Sleeps the internal drawing timer of a CartesianCanvas object.

A timer is put to sleep until a subsequent event is ready to occur.

The drawing timer is put to sleep until the next drawing frame is ready to occur.

Note
Identical to Canvas::sleep().

Referenced by tsgl::IntegralViewer::rectangleEvaluate(), and tsgl::IntegralViewer::trapezoidEvaluate().

Decimal  y,
Decimal  scale

Zoom the CartesianCanvas with a given center.

This function will re-center the CartesianCanvas at the given coordinates, then zoom with respect to the given scale.

Parameters
xThe coordinate to re-center the screen on.
yThe coordinate to re-center the screen on.
scaleThe zoom scale compared to the original. Less than 1 zooms in, greater than 1 zooms out.
Note
This function will automatically maintain the current aspect ratio.

Referenced by zoom().

Zoom the CartesianCanvas with the given bounding (Cartesian) coordinates.

This function will zoom the CartesianCanvas with respect to the given bounding coordinates.

Parameters
x1The left Cartesian bound.
y1The bottom Cartesian bound.
x2The right Cartesian bound.
y2The top Cartesian bound.
Note
Setting the right bound lower than the left bound or the top lower than the bottom will just swap the variables.
Warning
This function will NOT automatically maintain the previous aspect ratio.
Change the aspect ratio on-the-fly only with caution.
